Compétence 1 : Développer des application informatiques simples


Implémentation d'un besoin client

Un client nous demande un programme en langage C en tenant un compte rendu pour qu'il puisse comprendre.

On doit réaliser le du jeu du snake en langage C.

Réalisation du programme en 4 étapes continues et tenue d'un cahier de test permettant de rendre compte du bon fonctionnement du programme.

Possibilité de jouer au jeu dans le terminal de commande Linux ou le serpent doit manger 10 pommes pour gagner.


Développement d'une application

Réception d'un organigramme des tâches, d'une matrice RACI et d'un GANT pour créer une application qui s'occupe de l'emploi du temps des professeurs d'une école de danse.

En groupe, chacun doit se répartir les rôles de la matrice RACI et programmer la partie de l'application qui lui est dédiée en Javascript.

Compétence 2 : Appréhender et construire des algorithmes


Comparaison d'approche algorithmique

Découverte du code du jeu du snake de la part d'uun client qui nous demande de l'optimiser et de l'améliorer.

En 4 étapes, améliorer pas à pas le code de notre client en langage C d'après son cahier des charges.

Programmation en langage C tout en tenant un cahier de test permettant de vérifier le bon fonctionnement du jeu.

Le jeu finale ressemblait à une nouvelle version du jeu du snake entièrement automatisée avec 2 serpents qui finissent le jeu de la manière la plus optimisée possible.

Présentation des résultats des programmes pythons réalisés lors de notre présentation orale.

Compétence 4 : Concevoir et mettre en place une base de données à partir d'un cahier des charges client


Création d'une base de données

Une ligue de football cherche à créer une base de données permettant d'accéder plus simplement à des données spécifiques à son championnat.

On doit créer une base de données reliant des clubs de football, des équipes, des joueurs, des matchs, des championnats ou encore des équipes à partir d'un diagramme UML.

Création de la base de données avec le logiciel SQL Workbench.

Base de données finie permettant de faire des requêtes pour la ligue de football, client de ce projet, et ainsi accéder à des données beaucoup plus facilement qu'avant.


Exploitation d'une base de données

Une école souhaite faire une base de données pour pouvoir trier les élèves souhaitant s'inscrire à cette école

On doit dans un premier temps créer la base de données pui ensuite la peupler.

Création de la base de données d'après un schéma UML reliant les classes Étudiant, Candidat, Semestre, Inscription, Module, Programme... sur SQL Workbench puis peuplement de cette dernière grâce à l'importation d'un fichier .csv dans le logiciel.
